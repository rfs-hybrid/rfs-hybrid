# Hi, I'm Márcio 👋

**Software Engineer | Embedded Automotive Veteran | Systems Programming Enthusiast**

Welcome to my GitHub! I am a software engineer with a Bachelor's in Electrical and Computer Engineering and 5 years of professional experience in the Embedded Automotive industry. I am currently pushing the boundaries of my architectural knowledge as a student at **42 Porto**. 

My engineering philosophy is simple: **understand the system from the ground up.** Having started my career engineering core embedded software for automotive giants (Bosch, ZF, Continental), I developed a deep appreciation for strict memory safety, MISRA compliance, and defensive programming. Now, I am using the rigorous 42 curriculum to master UNIX systems architecture, with the ultimate goal of expanding into Full Stack Web Development.

---

## 🚀 My Journey: From Silicon to Systems

* 🔌 **The Foundation (Embedded Systems):** My professional background gave me hands-on experience dealing with strict memory constraints, hardware-level debugging, and writing highly efficient ANSI C code for critical automotive systems (Steering Angle Sensors, ECU-Tests).
* ⚙️ **The Architecture (42 Common Core):** I am currently mastering UNIX/Linux process management, concurrent threading, and complex data structures. I treat every 42 project as production software, aiming for the absolute maximum bonus score (the "125% Standard") to prove my architecture is bulletproof.
* 🔭 **The Next Frontier:** Having built a rock-solid foundation in C and low-level logic, my immediate next steps are mastering Object-Oriented Programming (C++) and building my fundamental knowledge of modern Web Development to eventually transition into a Full Stack role.
